## Props
- 특별히 정의한 일부 속성을 포함한 모든 **React ButtonElement** 의 **property** 를 props 로 받습니다.
- 특별히 정의한 속성은 아래와 같습니다.
- `children`: 버튼의 내용을 결정합니다.
- `color`: 버튼의 색상을 결정합니다.
- `size` : 버튼의 크기를 결정합니다.
- `shape`: 버튼의 모양을 결정합니다.
- `outline`: 버튼의 색상 채움 여부를 결정합니다.
<Props of={Button} />

## Shape
- `shape` 값에 따라 버튼의 모양이 달라집니다.
- default 값은 `rect` 입니다.
